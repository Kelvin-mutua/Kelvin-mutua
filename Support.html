Support - Sales Reporting System

Overview

This support page provides information on how to use the Sales Reporting System, including how to add sales, generate reports, and understand the data presented in those reports.

Using the Sales Page

The Sales Page allows you to input items sold and finalize sales transactions. Hereâ€™s how to use it:

  • Adding Items: You can either scan a QR code or type the item name in the input field. You can also select an item from the dropdown menu.
  • Payment Section: Select the payment method (Cash, Credit Card, Mobile Payment) and enter the amount received from the customer.
  • Finalize Sale: Click the "Finalize Sale" button to complete the transaction. This will store the sale data in local storage.

Generating Reports

Once you have finalized sales, you can view reports based on the sales data stored in local storage. The following report options are available:

  • Daily Report: This report shows all sales made on the current day. It includes the report number, date of sale, payment method used, items sold, and total amount.
  • Monthly Report: This report summarizes all sales made in the current month, providing similar details as the daily report.
  • Sales Summary Report: This report displays all sales data stored in local storage, giving a comprehensive overview of all transactions.

Report Details

Each report contains the following information:

  • Report Number: A unique identifier for each report generated.
  • Date of Sale: The date when the sale occurred.
  • Payment Method Used: The method of payment selected during the sale (Cash, Credit Card, Mobile Payment).
  • Items Sold: A list of items that were part of the transaction.
  • Total Amount: The total sales amount for that transaction.

FAQs

Q: What should I do if there is no sales data available?

A: If you see a message indicating that no sales data is available, ensure that you have finalized at least one sale on the Sales Page.

Q: Can I clear the sales data?

A: Yes, you can clear the local storage data through your browser's developer tools. However, this will permanently delete all sales records.
